# Learn GameChanger

Simple training for sports parents and scorekeepers.

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- Vercel-ready deployment

## Run locally
```bash
npm install
npm run dev
```

Then open <http://localhost:3000>

## What is built
- SEO-friendly landing page
- Hero section with clear positioning
- Problem / solution sections
- FAQ
- Interest form placeholder
- Disclaimer footer language

## Next recommended steps
1. Connect the form to Tally, Google Forms, Formspree, or a custom backend route.
2. Replace placeholder credibility copy with her name and a short bio.
3. Add brand colors, logo, and a real testimonial if available.
4. Deploy to Vercel.

## Deploy
If Vercel CLI is authenticated on this machine:
```bash
vercel
```

If GitHub + Vercel are both connected:
- push repo to GitHub
- import repo into Vercel
- deploy
